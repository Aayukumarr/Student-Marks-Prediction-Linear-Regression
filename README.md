# Student Marks Prediction using Linear Regression

This project uses a Machine Learning Linear Regression model to predict student marks based on study hours.

## Technologies Used
- Python
- NumPy
- Scikit-learn
- Matplotlib

## Project Steps
- Data Preprocessing
- Train-Test Split
- Model Training
- Prediction
- Error Evaluation (MAE, MSE, RMSE)
- Data Visualization

## Output
The model predicts marks based on study hours and visualizes the regression line.
